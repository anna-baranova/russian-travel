# Путешествия по России

 **Проектная работа по итогам курса Адаптивная верстка (Яндекс Практикум)**

## Используемые технологии

1. Построение сеток с помощью Grid.
2. Гибкий контейнер Flexbox.
2. Создание адаптивной верстки с помощью медиазапросов.
3. БЭМ-методология.
